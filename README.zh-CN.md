# ABAutoDeploy

[Readme](https://github.com/ABpasser/ABAutoDeploy)

**技术栈：**

> Nodejs/MidWay，Mysql/Sqllite，Docker/Docker-compose

**功能：**

- [] 保存项目（名称、路径）
- [] 实现文件压缩上传、服务器解压
- [] docker 自动重启

## 快速入门

### 本地开发

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ npm start
$ npm stop
```
