# ABAutoDeploy

[中文说明](https://github.com/ABpasser/ABAutoDeploy/blob/master/README.zh-CN.md)

**Technology Stack：**

> Nodejs/MidWay，Mysql/Sqllite，Docker/Docker-compose

**Function：**

- [] Save projects (name, path)
- [] Enable file compression upload, server decompression
- [] docker auto restart
